Qlik Connectivity Nodes for KNIME

The purpose of the "Qlik Connectivity" collection of nodes is to read and write qvx files. The QVX Reader node converts qvx data into a Knime data table. The QVX Writer node converts a Knime data table into qvx data

Features:

Read qvx files:

Read information from:

  • User's file system.
  • The internet.

Write qvx files:

The user can choose data types for each column, or use the default data format. The allowed data types are:

  • UNKNOWN: Floating point numbers are represented by "UNKNOWN" by default.
  • ASCII: Values are treated as text.
  • INTEGER: Values are treated as integers
  • REAL: Values are treated as numbers with a fixed number of significant digits. The nDecs field defines the number of decimal places.
  • FIX: Values are treated as numbers with a fixed number of decimals. The nDecs field defines the number of decimal places.
  • MONEY: Values are treated as money.
  • DATE: Values are treated as dates, with format M/D/YYYY
  • TIME: Values are treated as time, with format hh:mm:ss TT
  • TIMESTAMP: Values are treated as time stamps, with format M/D/YYYY hh:mm:ss TT
  • INTERVAL: Values are treated as time intervals, with format hh:mm:ss TT

Installation

You can install these nodes in KNIME 3.7.2 or above following these steps:

  • in KNIME, open Help/Install new software
  • Click top right button “Add”, and add the repository:
    • name: “Recent unstable contributions”
    • Location: http://update.knime.com/community-contributions/trunk
  • After clicking ok, select this “Recent unstable contributions” entry in the combo box,
  • Select the collection of nodes named “Spatial Processing Nodes”
  • Follow the next steps to install

Demo, examples

  • Qvx Reader - Titanic Example - "Titanic.qvx" is read and bar graphs are generated in Knime based on this data.
  • Qvx Writer - Feedback Example - "Feedback.csv" is read, then "Feedback.qvx" is generated. QlikView can be used to create data visualizations based on this qvx file.
  • Qvx Reader Test - Variety of simple use cases for the QVX Reader node
  • Qvx Writer Test - Variety of simple use cases for the QVX Writer node.
